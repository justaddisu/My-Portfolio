# My Portfolio

A modern, responsive personal portfolio website for showcasing full-stack development and UI/UX experience.

## Overview

This portfolio highlights:
- Full-stack development experience with JavaScript technologies
- University application development contributions
- Selected projects and stack-based filtering
- Contact form and accessibility-focused UI

## Features

- Responsive layout for desktop and mobile
- White and blue theme with light/dark mode support
- Project search and filter controls
- Experience timeline section
- Contact form with client-side validation
- Custom blue A favicon

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Web App Manifest support

## Project Structure

- `index.html` - Main page structure and content
- `styles.css` - Styling, theme variables, layout, and animations
- `script.js` - Interactions (filters, theme toggle, form validation, reveal effects)
- `manifest.webmanifest` - PWA metadata and icon setup
- `assets/images/` - Portfolio media and favicon assets

## Run Locally

Because this is a static website, you can run it in any of these ways:

1. Open `index.html` directly in a browser.
2. Or serve it with a local static server:

```bash
# Python 3
python -m http.server 5500
```

Then open: http://localhost:5500

## Customization

Update content in `index.html`:
- Hero and About text
- Project cards
- Experience entries
- Contact details and social links

Update look and feel in `styles.css`:
- Color variables in `:root`
- Typography and spacing
- Component styles

Update interactions in `script.js`:
- Project filtering logic
- Theme toggle behavior
- Form handling
